# 📍 Life Journey - Guilherme Oyakawa

This is an interactive timeline web application built with **Python** and **Streamlit**. It showcases my personal and professional trajectory, from my roots in Brazil to my current focus as a Data Analyst in Europe.

## 🚀 Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Visualization:** Streamlit-Timeline (based on KnightLab TimelineJS)
* **Deployment:** Streamlit Cloud

## 🌍 About the Project
As a professional with dual Brazilian and Portuguese citizenship, this project serves as a visual resume to demonstrate my technical skills in Python and my multicultural background.

## 🛠️ How to run locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run life_journey.py`
